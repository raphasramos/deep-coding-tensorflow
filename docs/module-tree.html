Module Hierarchy

  • autoencoder: Implementation of image autoencoder and its dynamics.
  • enums: File with the useful enums for the project
  • keras_custom: File containing custom operations and layers for keras models
  • parser: Class that parses the output of run_model.py
  • parse_output: Script to parse output of run_model.py
  • run_model: This script runs an autoencoder model.
  • partition_db: Script that receives a folder, maps all files in that folder and partition this dataset into training and testing parts.
  • patches_generator: Script that receives an image database and generates a patch version of this database
  • rename_imgs_in_folder: This script receives a folder and rename all image files in this folder.
  • resize_and_evaluate: Script that makes downsampling and upsampling on images and measures the statistics (psnr and bpp) in order to compare with the original images
