_save_jpeg_analysis(self)

source code 

Method that makes jpeg analysis in the current database. In the case of jpeg it's useful because the codec works with nominal quality, not one directly converted into psnr / bpp. This method saves all jpeg calculated metrics for each quality in a csv file.

_load_net_output_data(self, orig_path)

source code 

Function that reads the latents and images generated by the model.

Parameters:
  • orig_path (PosixPath) - Path containing the data generated by the model (images and latents).
Returns: tf.Tensor
Returns three tf.Tensors, respectively:
  1. Tensor representing the original img (tf.uint8 dtype).
  2. Tensor representing the reconstructed img (tf.uint8 dtype).
  3. Tensor representing the latents (tf.int8 if quantized_latent is True).

_calc_bpp(latent, pixels_num)
Static Method

source code 

Function that calculates the bpp considering the gzip. WARNING: it always supposes that the latent is integer.

Parameters:
  • latent (tf.Tensor) - The binarized latents generated by the network.
  • pixels_num (integer) - Number of pixels of original image.
Returns: float
bpp after compressing by gzip

_calc_jpeg_metrics(self, orig_path, bpp_ref)

source code 

Function that matches the jpeg metrics and save the corresponding images. It uses the csv generated by _save_jpeg_analysis function to match the correct quality.

Parameters:
  • orig_path (PosixPath) - Original path of the image.
  • bpp_ref (float) - bpp of the image reconstructed by the network.
Returns: float
numbers representing psnr, ssim, ms-ssim and bpp of the image compressed by jpeg2k.

_calc_jpeg2k_metrics(self, orig_path, bpp_ref)

source code 

Function that matches the jpeg2k metrics and save the corresponding images -> MUST INSTALL imagemagick: sudo apt install imagemagick. It takes the original image and generates a compressed version using jpeg2k in order to compare them.

Parameters:
  • orig_path (PosixPath) - Original path of the image.
  • bpp_ref (float) - bpp of the image reconstructed by the network.
Returns: float
numbers representing psnr, ssim, ms-ssim and bpp of the image compressed by jpeg2k.

_save_out_analysis(self, paths, bpps, metrics, codec_name)

source code 

This method saves a csv containing the analysis for all images wrt all metrics for each codec.

Parameters:
  • paths (list) - contains the path of each image.
  • bpps (list) - contains the bpp of each image.
  • metrics (list) - contains the visual metrics (psnr, ssim and ms-ssim) of each image.
  • codec_name (string) - string representing the codec.

parse(self)

source code 

Main method to parse the output. It calculates the metrics for each codec and saves the analysis to a csv file by calling _save_out_analysis.
